A suite of tools written in Python3 to help with monitoring all systems on board a motor or a sailing vessel.

Cross platform, one stop solution that helps with gathering all sorts of metrics available on board and records them into a digital logbook format for easy reporting and analysis

Download the latest version

Victron, NMEA0183 and GPS

Capture metrics from Victron systems, NMEA networks and external GPS dongles

Terminal Console / Web application

View your dashboard in a colourful terminal window or as a web application accessible from any device on your network

Excel, CSV & MySQL DB support

Log metrics to an excel, csv files or to a MySQL database instance for later processing

GPX support

Log trip gps records using the GPX file format

User configured notifications

Support for sound and email based notifications

Open Source & Free to Use

Code is fully open source under the MIT license and the application is free to download and use

Monitoring made easy

Configure the software with the desired metrics, alarms, notifications and session parameters and let it do the rest. Sessions can be: Triggered manually, split by a desired interval or triggered by the presence of one of the supported systems for convenience.
